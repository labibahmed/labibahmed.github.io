Hand Gesture Recognition Glove
Academic

Created a glove with flex sensors which read the input of each finger. The input of each finger is then sent over to a PC where a python script runs to recognize the gesture made. Gesture detection is done with supervised learning on the inputs of the glove and can be used to interpret sign language or to interact with video games.

Maze solving Robot
Academic

Programmed a small robot to follow and solve a maze. The robot itself had light sensors attached to it which is used to detect the lines of the maze. Using assembly as the language, my group and I created the program for which it takes in the input of the light sensors and then makes a decision to move. The program consisted of a input reader, state manager, and a distributer where all the logic for decision making was done.
